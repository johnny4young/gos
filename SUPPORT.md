# Support

Use the channel that matches the request:

- **Questions and setup help:** start a GitHub Discussion.
- **Reproducible bugs:** open a bug report issue.
- **Feature ideas:** open a feature request or start a Discussion if the design
  is still exploratory.
- **Security reports:** use GitHub Security Advisories as described in
  [SECURITY.md](SECURITY.md).

## What to Include

For install or rollback issues, include:

- `gos version`
- `gos doctor`
- `go version`
- OS, architecture, shell, and install method
- exact command and full output
- custom `GOS_INSTALL_DIR`, `GOS_CACHE_DIR`, or PATH changes

Please do not post secrets, tokens, private machine paths, or sensitive
vulnerability details in public issues or discussions.
